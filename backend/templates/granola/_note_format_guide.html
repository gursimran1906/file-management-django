Attendance notes — file number in the body

Notes saved in the “{{ config.attendance_folder|default:'Attendance Note' }}” folder. Put the matter file number in the note body (best set up as a Granola note template):

  • File number: ABC0010001 — links to matter ABC0010001, charged.
  • [ABC0010001] — bracketed form also works.
  • File number: ABC0010001 (NC) — links to ABC0010001, not charged.
  • No file number, or an unknown one → lands in the review inbox for manual assignment.

Free 30 minutes — parties template

Notes saved in the “{{ config.free30_folder|default:'Free 30 min' }}” folder. Add an “Info of Parties” section (best set up as a Granola note template) so attendee details are captured automatically: 

## Info of Parties
Party 1
- Name: John Smith
- Email: john@example.com
- Phone: 07123 456789
- Address: 1 High Street, London, Greater London, SW1A 1AA

Party 2
- Name: Jane Doe
- Email: jane@example.com
- Phone: 07900 000000

Each party starts at a Name: line. County and postcode are split out of a single Address: line automatically, or you can give County: / Postcode: explicitly.
