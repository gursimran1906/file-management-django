{% comment %}How fee earners should write their Granola notes. Shared by the all-staff guide page and the manager-only settings page.{% endcomment %}

Notes are recorded from one shared account, so add a Fee earner: line with your 3-letter staff code (e.g. Fee earner: ABC) to every note — otherwise we can't tell who ran the meeting and it waits in Allocate.

Attendance notes — file number in the body

Notes saved in the “{{ config.attendance_folder|default:'Attendance Note' }}” folder. Put the matter file number, your fee-earner code, and the start/finish times in the note body (best set up as a Granola note template):

  • File number: ABC0010001 — links to matter ABC0010001, charged.
  • [ABC0010001] — bracketed form also works.
  • File number: ABC0010001 (NC) — links to ABC0010001, not charged.
  • A note files automatically only when the file number, fee earner and start & finish times are all present; anything missing (or an unknown file number) → waits in Allocate → Meetings to be completed.

The start and finish times set the time recorded (units), so add them on their own lines: 

File number: ABC0010001
Fee earner: ABC
Start Time: 10:30
Finish Time: 11:15

A calendar event's times are used when there is one. Times like 9.30am also work. If no start/finish can be found the note waits in Allocate so the times can be added there.

Free 30 minutes — parties template

Notes saved in the “{{ config.free30_folder|default:'Free 30 min' }}” folder. Add your Fee earner: line, an “Info of Parties” section, plus the start and finish times (best set up as a Granola note template). A meeting is created automatically only when the parties and the fee earner are both present; otherwise it waits in Allocate → Meetings, where you add the missing details: 

Fee earner: ABC
Start Time: 14:00
Finish Time: 14:30

## Info of Parties
Party 1
- Name: John Smith
- Email: john@example.com
- Phone: 07123 456789
- Address: 1 High Street, London, Greater London, SW1A 1AA

Party 2
- Name: Jane Doe
- Email: jane@example.com
- Phone: 07900 000000

Each party starts at a Name: line. County and postcode are split out of a single Address: line automatically, or you can give County: / Postcode: explicitly.
